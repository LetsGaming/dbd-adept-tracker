# DBD Adept Tracker

Track your Dead by Daylight adept achievement progress for all Survivors and Killers. Syncs with Steam achievements, supports tier lists, stat cards, and progress sharing.

## Run it

```bash
npm install
npm run dev
```

Open `http://localhost:5173` in your browser.

## Usage

Select a character tab (Survivor / Killer / All), expand a character to mark progress, add tries, or mark as done. Connect Steam for automatic achievement sync.
